# react-aframe-demo
Intercomponental design demonstrated using react and a-frame.

## Installing

```
git clone https://github.com/KevinEverywhere/react-aframe-demo.git
cd react-aframe-demo
yarn  [OR]
npm install
```
## Running
```
yarn start
```
